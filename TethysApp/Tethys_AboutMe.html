Project Proposal: Estimating Erosion Rates of the Hawaiian Islands

The Gold Diggers



Overview

Erosion rates on the Hawaiian Islands are often studied and part of that process is estimating the mass flux of an island or part of an island. This can be done by estimating the amount of material that has left the system. The app would let the researcher do this quickly and efficiently.

Project Questions

What kind of vector and raster data analysis will be required?

The app would require several different vector and raster data analyses. The tools that would be needed are:

  • Clip
  • Interpolation
  • Zonal Statistics
  • Extract Data by Point
  • Raster Calculator
  • Flow Accumulation

What data will be distributed to the app via map services?

The primary data that will be distributed to the app via map services is DEM data. DEM data is common and should be easy to obtain for the Hawaiian Islands. The app will allow the user to specify which of the available datasets to use.

The app will also display a map of the Hawaiian Islands where the user can delineate an area to perform the analysis. The map provided to allow the user to specify the region of interest will be distributed by a map service.

What geoprocessing workflow will happen on the server?

The user will make points on the map of the highpoints (ridgelines) which will be used to extract the elevation at the points from the DEM of the island. Then these new points will be used to interpolate a new surface, the pre-erosional surface, of the island.The area of interest from the original DEM and the new surface will be clipped. These new clipped rasters are multiplied by the area of each cell so we know the volume of each cell. Then zonal statistics is used to find the total volume of each DEM. The raster raster calculator is used to find the difference and amount of volume lost in the chosen area. Mass flux can then be calculated using the age and density of basalt in the chosen area.

What are the Basic functions that will exist on the front-end user interface?

An interactive map of the Hawaiian Islands will be presented on the front-end user interface which will allow the user to scroll and zoom. The map will also allow the user to create points along the ridges of interest. The interactive map will look similar to that of the interactive map provided on the AGRC website (shown below). However, the interactive map created as part of this project will be significantly simpler.

There will also be input boxes where the user can enter the age and density of the basalt that will be used in the calculations. Once the area of interest has been defined, the user will be presented with a list of available DEM data, organized by the year the data was collected. The user will select the dataset they are interested in and the app will calculate the difference in volume between the selected data and the interpolated surface.

After the volume and mass flux has been calculated, the results will be shown to the user through a table that will pop up at the bottom of the app. The results will be printable so the user can export the results.
